1. Stop all containers in project in container manager
2. Delete dashboard container and image
3. Copy all new files in /src (not layout.tsx)
4. Build project
5. Correct errors
6. Connect in ssh to the NAS
7. Run the command: sudo docker exec -it dashboard npx prisma migrate deploy (only if there is some changes to the prisma schema)
